Visual Studio code

Visual Studio Code, commonly referred to as VS Code,[8] is an integrated development environment developed by Microsoft for Windows, Linux, macOS and web browsers.[9][10] Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add functionality.

VScode Features:

Language support: VS Code supports hundreds of languages, including JavaScript Syntax highlighting: Automatically highlights syntax to help you identify errors Bracket matching: Automatically matches brackets to help you identify errors Auto-indentation: Automatically indents lines of code Snippets: Provides code snippets to help you save time Debugging: Step through source code, inspect variables, and view call stacks Version control: Built-in support for Git and other source control providers

Shortcuts:

ctrl+c copy selected text
ctrl+v Paste the selected text
ctrl+s save the current file
ctrl+o opens the selected file
ctrl+x cut
ctrl+z cut the selected text or item
ctrl+y Redo the previously undone action
ctrl+F Find text in the current file
ctrl+A select all content in the current file
ctrl+H Replace text in the current file
ctrl+G go to line
ctrl+shift+N New window
ctrl+shift+W close window
ctrl+shift+P open command palette
ctrl+Shift+Alt+Down Add cursors to multiple lines for simultaneous editing
alt+Up Arrow Move the selected line or block of code one line up
Alt+Down Arrow Move the selected line or blockof code one line Down
